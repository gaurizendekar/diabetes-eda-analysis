# Diabetes EDA Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on a diabetes dataset to identify patterns, trends, and relationships among various health indicators. The analysis helps understand factors associated with diabetes through statistical summaries and visualizations.

## Objectives

* Understand the structure and quality of the dataset
* Perform data cleaning and preprocessing
* Analyze feature distributions
* Identify correlations between variables
* Detect outliers and anomalies
* Generate meaningful insights using data visualization

## Dataset Features

The dataset contains medical and demographic attributes commonly associated with diabetes, such as:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
* Outcome (Diabetic / Non-Diabetic)

## Technologies Used

* Python
* Jupyter Notebook

* Matplotlib

## Analysis Performed

### Data Preprocessing

* Handling missing values

### Exploratory Data Analysis

* Statistical summaries

* Histograms

## Key Insights

* Examined relationships between glucose levels, BMI, age, and diabetes outcomes.
* Identified significant correlations among health indicators.
* Visualized patterns that help understand diabetes risk factors.
